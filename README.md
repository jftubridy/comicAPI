## Description
* This is a program to check the weather of a user designated location.


## Technologies Used
* JavaScript
* JQuery
* HTML5
* CSS
* Bootsrap
* Webpack

## Installation
* Clone repo, from the root directory, install dependencies.

```sh
$ git clone https://github.com/jftubridy/comicAPI.git
$ cd comicAPI
$ npm install
```
